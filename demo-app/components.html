Light DOM — Bootstrap

Gebruikersprofiel

Naam: Jan de Vries

Rol: Administrator
Hoe werkt het?
  • Geen attachShadow()
  • Content via data-header en data-body
  • Bootstrap card, card-header werken direct
  • Alles zichtbaar in DevTools zonder shadow root

Shadow DOM — eigen CSS

Gebruikersprofiel

Naam: Sara Jansen

Rol: Redacteur
Hoe werkt het?
  • attachShadow({ mode: 'open' })
  • Content via slot="header" en default slot
  • Bootstrap bereikt de shadow root niet
  • DevTools toont aparte #shadow-root

Gebruik in HTML

<!-- Light DOM -->
<demo-card-light>
  <span data-header>Titel</span>
  <div data-body>
    <p>Inhoud</p>
  </div>
</demo-card-light>
<!-- Shadow DOM -->
<demo-card-shadow>
  <span slot="header">Titel</span>
  <p>Inhoud</p>
</demo-card-shadow>
