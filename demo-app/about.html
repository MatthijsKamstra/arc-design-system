Hoe deze app is gebouwd

Aanpak

Deze app gebruikt Bootstrap 5 voor stijl en layout, gecombineerd met Light DOM Web Components voor herbruikbare UI-elementen.

Geen framework. Geen build-stap. Gewoon de Web Platform APIs zoals die bedoeld zijn.

De <ui-nav> component rendert direct in de pagina-DOM. Bootstrap-klassen zoals navbar-dark en nav-link active werken zonder extra configuratie.

Waarom Light DOM hier de juiste keuze is
  • Bootstrap werkt direct — geen CSS hoeft opnieuw geïmporteerd te worden
  • De navigatie erft navbar-dark bg-dark zonder Shadow DOM-barrières
  • DevTools toont de gegenereerde HTML direct, geen verborgen shadow root
  • Het current-attribuut bepaalt welk nav-item active krijgt
